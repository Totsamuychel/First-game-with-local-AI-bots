# Многомировая 2D-Игра с AI Ботами

Это масштабная 2D-игра на Pygame, где вы собираете ресурсы, строите базу, исследуете подземелья и сражаетесь с AI-ботами, управляемыми локальной языковой моделью через Ollama.

## 🚀 Особенности
- **Три игровых мира**:
  - 🌲 **Мир Ресурсов**: Собирайте дерево, камень, железо, золото и кристаллы. Найдите тиммейтов и квесты.
  - 🏰 **Мир Базы**: Стройте стены, башни, казармы и магазины. Защищайтесь от набегов.
  - 👹 **Мир Подземелья**: Сражайтесь с монстрами за уникальные легендарные предметы и ресурсы.
- **Интеграция с Ollama**: AI-игроки анализируют ситуацию в реальном времени и принимают решения с помощью локальных LLM (рекомендуется `qwen2.5-coder:1.5b` или `gemma2:2b`).
- **Система прогрессии**: Повышайте уровень, улучшайте навыки сбора и боя, создавайте мощное снаряжение.
- **Сохранение прогресса**: Поддержка множественных сохранений в базе данных SQLite.

## 🛠 Установка

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/Totsamuychel/First-game-with-local-AI-bots.git
   cd First-game-with-local-AI-bots
   ```

2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Установите Ollama** (для работы AI):
   - Скачайте с [ollama.com](https://ollama.com/)
   - Запустите и скачайте модель:
     ```bash
     ollama pull qwen2.5-coder:1.5b
     ```

## 🎮 Запуск
```bash
python main.py
```

## 🕹 Управление
- **WASD / Стрелки**: Движение
- **ЛКМ**: Сбор ресурсов / Атака / Строительство
- **1-8**: Выбор постройки или оружия
- **TAB**: Открыть инвентарь
- **E**: Взаимодействие с порталами и объектами
- **ESC**: Меню / Выход

## 📂 Структура проекта
- `main.py`: Точка входа в игру.
- `game/`: Основной пакет с логикой игры.
  - `core/`: Ядро игры (игровой цикл, камера, ИИ).
  - `entities/`: Игровые сущности (игрок, враги, предметы).
  - `world/`: Объекты мира и здания.
  - `ui/`: Интерфейсы и диалоги.
  - `utils/`: Утилиты (база данных).
- `game/config.py`: Настройки баланса, цветов и константы.
