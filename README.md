# Многомировая 2D-игра: Ресурсы, строительство и PvP

## 🎮 Описание игры

Многомировая стратегическая игра в реальном времени с элементами сбора ресурсов, строительства базы и PvP сражений. Игра поддерживает ИИ противников на базе Ollama для более интеллектуального геймплея.

## 🚀 Установка и запуск

### Требования
- Python 3.7+
- pygame
- ollama (опционально, для умного ИИ)

### Установка зависимостей
```bash
pip install pygame ollama requests
```

### Установка Ollama (для умного ИИ)

1. **Скачайте Ollama** с официального сайта: https://ollama.ai/
2. **Установите Ollama** на вашу систему
3. **Запустите Ollama** в терминале:
   ```bash
   ollama serve
   ```
4. **Скачайте модель** (в другом терминале):
   ```bash
   ollama pull llama3.2
   ```

### Запуск игры
```bash
python main.py
```

## 🎯 Игровые механики

### Два мира
- **Мир ресурсов** (PvP отключен) - безопасная зона для сбора ресурсов
- **Мир базы** (PvP включен) - зона строительства и сражений

### Ресурсы
- 🌳 **Дерево** - базовый строительный материал
- 🪨 **Камень** - для укреплений
- ⚙️ **Железо** - для оружия и продвинутых построек
- 💰 **Золото** - для элитного снаряжения
- 💎 **Кристаллы** - для магических предметов

### Постройки
- **Стены** - защита базы
- **Башни** - автоматическая атака врагов
- **Казармы** - создание тиммейтов
- **Мастерская** - создание оружия и брони
- **Порталы** - перемещение между мирами

### Боевая система
- **4 типа оружия**: меч, лук, топор, посох
- **4 типа брони**: кожаная, железная, золотая, кристальная
- **Система здоровья** с регенерацией
- **Тиммейты** - союзники для помощи

## 🎮 Управление

### Главное меню
- **Стрелки вверх/вниз** - навигация по меню
- **Влево/вправо** - изменение количества ИИ противников (1-4)
- **Enter** - выбрать пункт меню

### В игре
- **WASD** - движение игрока
- **E** - взаимодействие с порталами
- **Пробел** - переключение режимов (исследование/строительство/бой)
- **ЛКМ** - действие (строительство/атака/сбор)
- **1-5** - выбор типа постройки
- **TAB** - открыть/закрыть инвентарь
- **S** - сохранить игру
- **L** - загрузить игру
- **ESC** - вернуться в главное меню

## 🤖 ИИ противники

Игра поддерживает два типа ИИ:

### Умный ИИ (с Ollama)
- Использует языковую модель для принятия решений
- Анализирует игровую ситуацию
- Адаптируется к стратегии игрока
- Принимает сложные тактические решения

### Простой ИИ (резервный)
- Базовая логика без внешних зависимостей
- Простые правила поведения
- Работает всегда, даже без Ollama

## 📁 Система сохранений

- Автоматическое сохранение при выходе из игры
- Ручное сохранение клавишей **S**
- Сохраняется:
  - Позиции всех игроков
  - Ресурсы и инвентарь
  - Постройки и их состояние
  - Тиммейты и их характеристики
  - Текущий мир и режим игры

## 🎯 Стратегии игры

### Для новичков
1. Начните с исследования мира ресурсов
2. Собирайте ресурсы и ищите тиммейтов в кучах
3. Постройте мастерскую для создания оружия
4. Укрепите базу стенами и башнями

### Для опытных игроков
1. Быстро развивайте экономику ресурсов
2. Создавайте армию тиммейтов через казармы
3. Используйте разведку для атак на вражеские базы
4. Контролируйте ключевые точки на карте

## 🔧 Настройка

Все настройки игры находятся в начале файла `main.py`:
- Скорость игрока и ИИ
- Стоимость построек
- Характеристики оружия и брони
- Частота появления ресурсов
- Размеры мира и баз

## 🐛 Известные проблемы

- При отсутствии Ollama ИИ использует простую логику
- Некоторые модели Ollama могут работать медленно
- Большое количество ИИ может снизить производительность
